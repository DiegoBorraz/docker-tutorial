Tutorial Docker basico

1- Começo a partir da imagem do nginx

docker run -p 8080:80 nginx

Aqui eu digo que ao rodar o nginx no meu container a porta 80 do Container estará apontada para a 8080 da minha maquina

2- Aqui eu digo que tudo que esta na minha pasta html altera em tempo real a pasta html do meu container

docker run -p 8080:80 -v $(pwd)/html:/usr/share/nginx/html nginx

3- Aqui eu copio o index.html para dentro do nginx

COPY html/index.html /usr/share/nginx/html/

4- Agora buildo o docker e dou um nome a ele

ocker build -t diegoavila33/docker-tutorial:latest .

5- Agora subo o container para o DockerHub

docker push diegoavila33/docker-tutorial

6- Para pegar o container

docker run -p 8080:80 diegoavila33/docker-tutorial
